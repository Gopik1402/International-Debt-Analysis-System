# International-Debt-Analysis-System
International Debt Analysis System

**Project Overview**

This project is an end-to-end data engineering and business intelligence solution analyzing the World Bank's International Debt Statistics. It tracks the external debt of developing nations from 2000 to 2024, uncovering historical trends, indicator distributions, and extreme debt burdens to support data-driven policy decisions.

**Tech Stack**

Database: MySQL, MySQL Workbench

Data Processing: Python, Pandas, SQLAlchemy

Visualization: Plotly Express (Python), Power BI (DAX, Power Query)
